= inst

Common MIDI instrument/sequencer functionality

Used by {diamond}[http://github.com/arirusso/diamond] and others in progress

Requires {midi-message}[http://github.com/arirusso/midi-message] which should install automatically with the gem

== Installation

gem install inst
